# Automated Fairness Auditor AI

This project detects and mitigates bias in machine learning models.

## Problem
ML models can be biased in decision-making systems like loan approvals.

## Features
- Train ML model (Logistic Regression)
- Evaluate performance
- Detect bias using Fairlearn
- Apply bias mitigation
- Automated fairness pipeline

## Tech Stack
- Python
- scikit-learn
- Fairlearn
- Google Colab

## Dataset
Adult Income Dataset (UCI)

## Solution
This project detects and mitigates bias using Fairlearn.

## Model Comparison
Logistic Regression:
- Accuracy: 0.807
- DP Difference: 0.123

Random Forest:
- Accuracy: 0.856
- DP Difference: 0.172

## Key Insight
Random Forest achieved higher accuracy but showed higher bias compared to Logistic Regression. This demonstrates the tradeoff between performance and fairness in machine learning models.
