# Automated Fairness Auditor AI

This project detects and mitigates bias in machine learning models.

## Features
- Train ML model (Logistic Regression)
- Evaluate performance
- Detect bias using Fairlearn
- Apply bias mitigation
- Automated fairness pipeline

## Tech Stack
- Python
- scikit-learn
- Fairlearn
- Google Colab

## Dataset
Adult Income Dataset (UCI)
